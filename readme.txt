== Description ==

This plugins had the needed code to the header of your blog, to detect your OpenID account at http://openid.sapo.pt.
Configure your person name at the administration panel.
Look at my blog source code and check the header (http://antoniocampos.no-ip.com).

== Installation ==

1. Upload the folder "SapoOpenID" to the "/wp-content/plugins/" directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to settings and insert your persona name (created at http://openid.sapo.pt)
